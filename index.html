1. Recap of background:

1 Million Women is a medium-sized non-profit organisation based in Australia, it is founded by Natalie Isaacs in 2006. The main goal of this organisation is calling upon women and girls all over the world to tackle the climate crisis through a lifestyle revolution.

  • (1 Million Women, 2015)

3. Development from proposal:

Based on the mood board, user profile, personas from the proposal and gathered feedback from user testing, this website prototype is developed using HTML and CSS to achieve the main goal of 1 Million Women organisation as encouraging climate actions. The main visual style and structure remain the same to represent the identity of the organisation, but minor iterations are made to provide a better user experience and usabilities.

3.1. Changes in new iteration:

Among various proposed subpages, the home page and the other 3 subpages, including About Us, Join and Donate, can provide a clear call-to-action for users to join the organisation while educating the values of climate actions.

Key words: Consistency & Simplicity

3.1.1. Navigation and footer:

  • While the initial proposal used a static navigation bar on the top of each page and a thick footer with an icon and related links, the final prototype used a sticky navigation bar and a simpler footer without the icon for simplicity.
  • For the mobile interface, the responsive navigation bar retains the menu icon and organisation icon and the “donate” and “join” button will show by clicking on the menu icon.

3.1.2. Form:

  • The inline input on the desktop interface will wrap to the next line if the screen size is smaller, therefore the user can fill the form more easily on mobile devices.

3.1.3. Home page heading:

  • Instead of overlapping the heading on the line, now the gradient-coloured horizontal line can divide the home page into different sections more clearly.

3.1.4. Colour palette:

  • Some visual styles have been changed for higher color contrast ratios, therefore to improve the level of accessibility for visually impaired people: e.g. text colour of buttons, background color of text.

3.1.5. Shape:

  • Use rounded and organic shapes can build focus, positivity, grab attention and is easier for visual processing(Pumpkin Web Design Manchester, 2020): e.g. buttons, images and article containers.

3.1.6. Content:

  • Call-To-Action: Use engaging images and positive tones of speaking: e.g. "Will you join our movement?" "Are you with us?" "Join us and get counted with the other 832,454 women"
  • Education: Use an interactive website sourced from NASA to inform the impacts of climate change

3.1.7. Transition:

  • Buttons styles will change when hovering and give visual feedback:
    • Text colour of the buttons on the landing page will change into white;
    • Background colour of the “Learn more” buttons will change into tiffany blue(#04BFB3).
  • Background colour of inputs in focus will change from white to platinum grey(#E3E3E3) and show the state has been changed as selected.
  • Border colour of the donation plan will change into dark green(#4E6C6A) when hovering, or tiffany blue(#04BFB3) when being selected. And the background colour of the selected options will change into black(#000000) while text colour changes as Tiffany blue(#04BFB3)

3.1.8. Cursor status:

  • The conceptual model of pointer cursor is a perceptible affordance that signifies the element is clickable. On the contrary, the default cursor signifies this element could be selected but not clickable.

4. Annotations for new features:

4.1. Home page:

Landing page:

  • Rather than using a monochrome photo, the new iteration uses a dark transparent mask on the original photograph to retain the lively colour and exemplify the optimum attitude of women power.
  • Due to the sticky navigation bar, the navigation on the landing page would be no longer necessary. Therefore, by adding a “learn” button, the home page will smoothly scroll down to the main content.

Main content:

  • Content for “About Us” section has been reduced for simplicity on the home page and encourages user to click on the “Learn more” button for more details.
  • The multiple-sized photos displayed in an image gallery is unnecessarily complex to use. Therefore, the iterated “Take Actions” section uses images in a consistent size and is animated as an auto-slideshow.

4.2. Join page:

The final prootype adds a new option of 'Country' to the form: 1 million women is a global organisation that therefore, it is necessary to collect the information about which country the user is living in.

4.3. Donate page:

The initial proposal used a progression bar to demonstrate steps of proceeding the donation. However, the final prototype used direct headings assisted with “Back” ”Next” “Previous” and ”Proceed” buttons to amplify the discoverability and offer an intuitive user experience.

4.4. About Us page:

To improve accessibility, background colour of article cards changed to dark slate grey (#345151) for higher contrast.

5. Overall flow:

The home page provides with 3 buttons for “Join” “Donate“ as call-to-action and “Learn” as eduction to invite users to browse the main content on the home page. The main content includes educational contents such as statistics of climate change with an interactive Climate Time machine iframe from NASA in “About Climate Change” section, a brief introduction to the organisation in “About Us” section, and further get back to the call-to-action contents of “Take Actions” and “Donating” sections.

About Us page provides information about the founder, team and partners to improve the trustworthiness of 1 Million Women, therefore learning more about the values of “women power” and “climate actions “ behind the organisation and also encouraging people to join in as a member.

Join page encourages users to join using the statistics of the current member and provides a form for a simple and quickly registration including “First name”, “Last name”, “email” and “password”, as well as a dropdown menu for “country” to collect valuable information that could possibly be interesting to the organisation and stored at back-end system.

Donate page provides 3 steps of donating to the organisation, the purpose of each step is highlighted in the heading and explained in the paragraphs underneath. Step1 requires the user to choose a donation plan, it could be either once-off, weekly and monthly donation with a suggested amount of money but also provides flexibilities for uses to choose a preferred plan. Step2 asks users to leave their names and contact information as records without too much obstructive content. Step3 is a payment gateway that allows users to choose a payment method and fill in the bank details for proceeding.

6. Accessibility testing:

Accessibility is a practice of making the website available and readable for more groups. 1 Million Women is a global organisation that aims at all females which include different abilities to access the website, therefore accessibility is an essential factor that needs to be considered during the process of web development.

This website is evaluated by WAVE, an online accessibility tool, the overall result is optimum.

  • (WAVE,2020)

7. Usability testing (SUS):

System Usability Score(SUS) test is a widely used tool for evaluating usabilities, an average score of 68 is used as the standard of industries.

After gathering multiple responses form participants, the average score of the final prototype is 83.75 > 68.
